# 창세기전3 세이브 에디터 (파트1 · 파트2 통합)

[창세기전3](https://namu.wiki/w/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%84%203) **파트1·파트2**의 세이브 파일을 브라우저에서 편집하는 도구입니다.
설치가 필요 없고, **세이브 파일은 전부 브라우저 안에서만 처리**되어 어디에도 업로드되지 않습니다.

> ### ▶ 바로 사용하기 (설치 불필요)
> **<https://hyperize0807.github.io/genesis3-editor/>**
> PC·모바일 모두 위 주소를 브라우저로 열면 바로 사용할 수 있습니다.

> ⚠️ **편집 전 세이브 파일을 반드시 백업**하세요. 잘못된 편집은 게임 진행에 영향을 줄 수 있습니다.

---

## 사용 방법

1. 첫 화면에서 **파트1 / 파트2 로고**를 클릭해 작품을 선택합니다.
   - 또는 하단 상자에 **세이브 파일을 끌어다 놓으면** 작품을 자동 판별해 바로 엽니다.
2. **📂 .sav 열기**(또는 드래그)로 세이브 파일을 불러옵니다.
3. 원하는 항목을 편집합니다.
4. **저장(다운로드)** 을 누르면 `..._edited.sav` 파일이 내려받아집니다.
5. 백업 후, 내려받은 파일의 이름을 원래 세이브 이름으로 바꿔 게임 폴더에 덮어씁니다.
6. 게임에서 정상 로드되는지 확인합니다.

> 저장 시 게임의 변조방지 체크섬이 자동으로 다시 계산되어 정상 로드됩니다.
> "← 작품 선택"으로 언제든 첫 화면으로 돌아갈 수 있습니다.

## 🔢 지원 버전 (중요)

| 작품 | 세이브 위치(예) | 지원 버전 |
|---|---|---|
| **파트1** | `<게임폴더>\g30001.sav` ~ | **1.03+ / 1.04** (열 때 자동 감지, 저장 버전 선택 가능) |
| **파트2** | `<게임폴더>\G3P_II00.sav` ~ | **게임 버전 1.005 전용** |

> ⚠️ **파트2는 현재 게임 버전 1.005 세이브만 편집할 수 있습니다.**
> 널리 쓰이는 **1.022 등 다른 버전은 아직 지원하지 않으며, 추후 추가 예정**입니다.
> 1.005가 아닌 버전의 세이브는 편집하지 마세요(파일이 손상될 수 있습니다).

## 편집 가능한 항목

### 파트1
- 💰 **소지금**(현재 에피소드)
- 🧑 **캐릭터**: 소속·직업·초상화, 능력치(HP·EXP·STR~WTP), 장비 5슬롯, 어빌리티(레벨·추가/제거, 최대레벨 자동 적용)
- 🎒 **보관함**(현재 에피소드 안전 편집 / 그 외 위험 영역)
- 📖 **아이템 사전** · 📜 **직업 사전** · 📘 **전직 공략**

### 파트2
- 💰 **소지금** · 🎒 **인벤토리**(상단 버튼, 추가/삭제·수량·종류)
- 🧑 **캐릭터**: 직업·체질(스타일), 레벨, EXP(누적/현재), 능력치(LP·PSY·CTP·RDP·DEX), 장비 6슬롯, 어빌리티, 기본 능력치(영구 강화)
  - 레벨은 **누적 EXP**로 정해집니다. 레벨을 바꾸면 누적 EXP가 자동 설정되고, **전투에 진입하면** 능력치가 소급 반영됩니다.
  - HP·ATK·ACR·STP는 게임이 계산하는 값이라 직접 편집 항목이 아닙니다.
- 📖 **아이템 사전**: 이름·설명·ATK/HP 배수·능력치 보정·판매처·가격(상점별). 이름/설명/상점명으로 검색.
- ⚔ **전투 중 세이브**는 자동 감지해 안내합니다. 출전 중인 캐릭터는 전투가 끝난 뒤 반영되므로, 가급적 비전투(마을/필드) 세이브에서 편집하세요.

## 자주 묻는 질문

**인터넷 연결이 필요한가요?** 아니요. 완전히 오프라인으로 동작하며 세이브 파일은 PC 밖으로 나가지 않습니다.

**어떤 브라우저를 쓰나요?** 최신 크롬·엣지·파이어폭스·사파리 등에서 동작합니다.

**📱 모바일(iOS)에서 버튼이 안 눌려요.** iOS '파일' 앱에서 HTML을 직접 탭하면 미리보기(Quick Look)로 열려 자바스크립트가 제한됩니다. 반드시 위 **웹 주소를 브라우저로** 여세요.

**게임이 "허가없이 변경되었습니다"라며 로드를 거부해요.** 이 에디터로 저장한 `_edited.sav`인지, 파트2라면 **게임 버전이 1.005인지** 확인하세요.

## 내려받아 쓰기 (오프라인)

[`index.html`](index.html) **한 파일만** 내려받아 브라우저로 열면 됩니다.
두 작품 에디터와 로고 이미지까지 **모두 이 파일 하나에 포함**되어 있어, 다른 파일은 필요 없습니다.
